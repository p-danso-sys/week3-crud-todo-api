# Week 3 - CRUD Todo API

RESTful API for managing todos with full CRUD operations.

## Features

- ✅ Create todos with validation
- ✅ Read all todos
- ✅ Read single todo by ID
- ✅ Update todos
- ✅ Delete todos
- ✅ Bonus: Get active todos

## Tech Stack

- Node.js
- Express.js
- JSON file storage

## Installation

```bash
npm install
npm run dev
